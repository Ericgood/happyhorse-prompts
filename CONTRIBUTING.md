# Contributing to HappyHorse Prompts

Thank you for your interest in contributing! 🐴

## Ways to Contribute

### 1. Submit a Prompt (Recommended)

Create a Markdown file in the appropriate `prompts/<category>/` folder:

```
prompts/
├── cinematic/     # Movie-style, dramatic scenes
├── nature/        # Wildlife, natural landscapes
├── abstract/      # Artistic, experimental visuals
├── sci-fi/        # Futuristic, cyberpunk themes
├── landscape/     # Scenic environments
└── macro/         # Extreme close-ups, details
```

Use this template:

```markdown
---
title: Your Prompt Title
author: Your Name
author_link: https://twitter.com/yourhandle
model: HappyHorse Pro
category: cinematic
video_url: https://...
date: 2026-04-09
---

## Description

Brief description of the generated output.

## Prompt

\```
Your prompt text here...
\```

## Preview

![Preview](url-to-preview-image)
```

### 2. Open an Issue

If you'd rather not submit a PR, [open an issue](https://github.com/Ericgood/happyhorse-prompts/issues/new) with:

- Your prompt text
- The model used
- A description of the output
- Video/image preview (if available)

### 3. Suggest a New Category

If your prompt doesn't fit existing categories, suggest a new one in your PR or issue.

## Guidelines

- ✅ Share original prompts or ones you have permission to share
- ✅ Include the model name (e.g., HappyHorse Pro)
- ✅ Add a clear description
- ✅ Attach video/image previews when possible
- ❌ No NSFW content
- ❌ No copyrighted character reproductions
- ❌ No spam or low-effort submissions

## Pull Request Process

1. Fork the repo and create your branch from `main`
2. Add your prompt file(s) following the template
3. Ensure your Markdown is properly formatted
4. Submit the PR with a clear title

We typically review PRs within 48 hours. Thanks for contributing! 🎉
