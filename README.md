## 🤝 How to Contribute

We welcome prompt contributions from everyone! Here's how:

### Submit via Pull Request

1. **Fork** this repository
2. **Add your prompt** — create a Markdown file in the appropriate `prompts/<category>/` folder using the template below
3. **Submit a PR** — we'll review and merge it!

### Prompt Template

Create a file like `prompts/cinematic/my-awesome-prompt.md`:

```markdown
---
title: Your Prompt Title
author: Your Name
author_link: https://twitter.com/yourhandle  # optional
model: HappyHorse Pro
category: cinematic
video_url: https://...  # optional, link to generated video
date: 2026-04-09
---

## Description

A brief description of what this prompt generates.

## Prompt

\```
Your full prompt text here...
\```

## Preview

<!-- If you have a video or image preview, add it here -->
![Preview](url-to-image)
```

### Submit via Issue

Don't want to deal with PRs? Just [open an issue](https://github.com/Ericgood/happyhorse-prompts/issues/new) with your prompt and we'll add it for you!

### Guidelines

- ✅ Original prompts you created or have permission to share
- ✅ Include the model used (e.g., HappyHorse Pro)
- ✅ Add a description of the expected output
- ✅ Video/image previews are highly encouraged
- ❌ No NSFW content
- ❌ No copyrighted character reproductions
